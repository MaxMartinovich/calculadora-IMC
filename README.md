# Calculadora IMC

## Descrição
Aplicação simples para calcular o IMC
O aplicativo permite que o usuário adicione nome, peso e altura e ao clicar no botão o site já realiza o calculo

## Tecnologias utilizadas
- HTML
- CSS
- JS

## Funcionalidades
- Adicionar o nome
- Adicionar o peso
- Adicionar a altura
- Calculo com as exibições atualizadas da OMS sobre o IMC
- Inteface simples

## Como executar o projeto
1ª - Instale um IDE, de preferência o VSCode porque roda muito bem com HTML,CSS e JS.

2ª - Copie o código disponibilizado

3ª - Execute a aplicação!
